# 🐾 **PawMap**
Discover dog-friendly places around you

PawMap is an iOS app built with SwiftUI, MapKit, and Firebase, designed to connect dog owners through a shared map of pet-friendly cafés, parks, and stores.
Users can explore nearby spots, leave reviews, upload photos, and share recommendations — creating a community-driven guide for every dog lover. 🐕🌳

## ✨ **Features**

🗺️ Interactive Map — Discover dog-friendly cafés, parks, and stores near you.

🐕 User-Generated Content — Add new places, leave ratings, and upload photos.

⚡ Real-Time Updates — Powered by Firebase Firestore for instant sync across all users.

🔐 Secure Authentication — Sign in with Apple, Google, or Email.

📸 Photo Uploads — Store and display community-contributed images.
